# beatrizuezu.github.io

I used Jekyll and the theme is [Indigo theme](https://github.com/sergiokopplin/indigo)

### How to install

```shell
bundle install
```

### How to run
```shell
bundle exec jekyll serve --config _config.yml,_config-dev.yml
```

### How to navegate
* `http://127.0.0.1:4000`
* `http://127.0.0.1:4000/admin`
